PANAL
Una página web de almacenamiento de proyectos de programación en los lenguajes de HTML, CSS y JavaScript y una interfaz para probar dichos proyectos. Este sitio está orientado al desarrollo de proyectos de programación principalmente para personas principiantes o profesionales que necesiten un acceso rápido a herramientas de compilación desde cualquier lugar. También se incluye distintos planes de trabajo según la necesidad del usuario, así como un área para guardar, compartir, modificar y descargar los proyectos generados por los usuarios.

WASAP
Es una aplicación de mensajería instantánea para teléfonos inteligentes que permite a los usuarios enviar mensajes de texto, así como imágenes, videos, documentos y mensajes de voz. SPOTIFY
